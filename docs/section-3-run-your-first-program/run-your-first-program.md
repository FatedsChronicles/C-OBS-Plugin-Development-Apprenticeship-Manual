# Run Your First Program

This is the magical moment.

#### Step 1 — Press **Ctrl + Shift + P**

A search box opens at the top.

#### Step 2 — Type:

```
CMake: Select a Kit
```

Press **Enter**.

#### Step 3 — Choose:

```nginx
MSVC x64
```

#### Step 4 — Build your program

Press **F7** on your keyboard.

VS Code will now compile (build) the program.

If it succeeds, you’ll see a message like:

```nginx
Build finished successfully
```

#### Step 5 — Run your program

In the bottom panel, click:

```
▶ Run
```

Or press:

**Ctrl + F5**

You should see:

```
Hello, world!
```

If you saw the message — you just wrote your first real program.
